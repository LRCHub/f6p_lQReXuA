# 中瀬ミル – 遠く、遠く、遠く。

> **歌詞登録ステータス：プレーン歌詞のみ**

```
きっとこの世界で
ずっと覚えてるのは
僕を僕たらしめる
あの頃のあなたで
わざわざさよならで
区切るほどじゃ無いから
いつのまにか僕らは
思い出に変わっていた
当たり前のように
みんな強くなっていく
知らない人と話す
ことも上手くなってる
そういえば、あなたは
ここにいない、あなたは
僕に話してくれた
幸せを覚えてる？
何気ない誰かのふざけた声が
あなたを攫っていった
当たり前に侵されていった
なのにこの声は終わらなかったから
あなたがくれた言葉だけが
いつまでも変わらないまんま
絶え間なく反芻するんだ
何故かずっと新しくなっていくんだ
叶うなら、あなたを忘れて遠い場所へ
なんて思いながら、動けない事の美しさを
誰かが描く、んでまた誰かが書く
その全てを僕ら本当に愛してた
あといくつ何を終わらせたら
あなたを知れるの
今その目に映る世界は
いつか僕らが拒んだ未来だった
白々しい顔でどうせ終わる
愛とか育んで
"当たり前"に注力するんだろう
踏み潰した妥協を誤魔化すように
「あの人は昔が良かった」
あなたもいつかそう言ってた
僕はずっとひとつ言いたくて
たまらないのにわからなかった
あなたが居た頃よりずっと
複雑で繊細になったよ
希望だった芸術がいつか
絶望に変わるのも知ったよ
何気なくあなたが語った夢が
あなたを攫ったまんま
当たり前に絆されていった
どこまでも綺麗だなんて思った
あの痛みの意味の
ひとつひとつを、終わらせたくはなかったから
形にして残していたんだろう
僕の感じ方なんてひとつも知らずに
馬鹿げた夢を語るあなたは輝いてた
ほつれうねった髪を束ねてひとつずつ
誰にも言えないその澱み濁った話を
またいつか聞かせてね僕は変わらないから。
```
